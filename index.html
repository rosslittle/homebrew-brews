What is this?.

A personal homebraw tap for mac. Feel free to use it but, of course, you do so at your own risk and I make no guarantee about the quality of the content in any way.

To Add this tap run the following command:

$ brew tap rosslittle/brews
$ brew install <formula>

That's it really... If a formula conflicts with one on the master or another tap, then you can install by specifying the URL to the file directly, or using:


$ brew install rosslittle/brews/<formula>
